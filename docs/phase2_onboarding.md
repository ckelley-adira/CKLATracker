# Phase 2 Onboarding Guide — CKLA Skills Tracker

Quick-start guide for teachers using the Phase 2 input automation features.

---

## What's New in Phase 2

Phase 2 adds four key features to reduce manual entry and speed up your workflow:

| Feature | What It Does | How to Access |
|---|---|---|
| **Navigation Hub** | One-click links to every tab, view, and tool | First tab in the workbook |
| **Google Form Entry** | Submit scores via a Google Form instead of editing cells | CKLA Tools → Phase 2 Tools → Form Manager |
| **Teacher Input Sheets** | A personal, lightweight sheet with just your students | CKLA Tools → Phase 2 Tools → Create Teacher Input Sheet |
| **Unit Tab Generator** | Create new unit tabs from a standard template | CKLA Tools → Phase 2 Tools → Generate New Unit Tab |

## Getting Started

### 1. Open the Navigation Hub

The **Navigation Hub** is the first tab in your workbook. It has clickable links to:
- Every unit assessment tab (organized by grade)
- Roster Views and Summary Charts
- Quick reference for all menu tools

> **Tip:** If the Navigation Hub is missing, go to **CKLA Tools → Phase 2 Tools → Build Navigation Hub** to create it.

### 2. Enter Scores via the Sidebar (Recommended)

This is the fastest way to enter scores for individual students:

1. Click **CKLA Tools → Enter Assessment Scores**
2. Select your **Grade**, **Unit**, and **Teacher name**
3. Select a **Student** from the dropdown
4. Enter scores in the form — it shows you the max points for each question
5. Click **Submit Scores** — the form auto-advances to the next student

### 3. Enter Scores via Google Form (Alternative)

For situations where you want a shareable link (e.g., entering scores from a mobile device):

1. Ask your admin to create a form: **CKLA Tools → Phase 2 Tools → Form Manager → Create Form**
2. Open the form link — it works in any browser
3. Enter the student name, your name, and scores
4. Scores are automatically imported into the correct unit tab

### 4. Use a Teacher Input Sheet (For Bulk Entry)

If you prefer editing a spreadsheet but don't want to navigate 46+ tabs:

1. Go to **CKLA Tools → Phase 2 Tools → Create Teacher Input Sheet**
2. Select your grade, unit, and name
3. A new tab appears with **only your students** and score columns
4. Enter all scores in this lightweight sheet
5. When done, go to **CKLA Tools → Phase 2 Tools → Sync Teacher Input Sheets**
6. Your scores are written back to the master unit tab

> **Important:** Always click **Sync** before the admin runs **Clean Up Input Sheets**, or your scores will be lost.

## Frequently Asked Questions

**Q: Do I need to change anything about how I read reports?**
A: No. Roster Views, Summary Charts, and all existing reports work exactly the same. Phase 2 only changes how scores are *entered* — all existing formulas continue to auto-calculate.

**Q: Can I still type scores directly into the unit tabs?**
A: Yes. The sidebar form, Google Forms, and teacher input sheets are optional — they're faster and less error-prone, but direct cell editing still works.

**Q: What happens if I enter the wrong score?**
A: Use the sidebar form to re-enter scores for that student — it will overwrite the previous values. Or edit the cell directly in the unit tab.

**Q: Who creates the Google Forms and Teacher Input Sheets?**
A: Your admin or data team lead. Teachers use the forms/sheets; admins manage them.

## Quick Reference: Phase 2 Menu

All Phase 2 tools are under **CKLA Tools → Phase 2 Tools**:

| Menu Item | Who Uses It | What It Does |
|---|---|---|
| Build Navigation Hub | Admin | Creates/refreshes the Navigation Hub tab |
| Form Manager | Admin | Creates Google Forms linked to unit tabs |
| Create Teacher Input Sheet | Admin / Teacher | Generates a personal input sheet |
| Sync Teacher Input Sheets | Admin | Writes input sheet scores back to master |
| Clean Up Input Sheets | Admin | Deletes all temporary input sheets |
| Generate New Unit Tab | Admin | Creates a new unit tab from template |
